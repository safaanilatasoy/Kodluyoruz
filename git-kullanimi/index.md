

# Yeni Bit Git Projesi Oluşturmak

```
git init
```

# Değişiklik Yapılan Dosyaları Gösteren Komut

```
git status
```

# Dosya Eklemek


```
git add .
```


# Commit İşlemi

```
git commit -m main
```


# Commitlediğimiz Dosyaları En Son Sunuzuya Kaydetmek İçin Kullanılan Komut

```
git push

```

# Commitlediğimiz Değişiklikleri Gösteren Komut

```
git log
```
